Coffee Machine

Preparation

Questions

  1. Have you used this machine before?
  2. Why might a coffee maker be useful to you?
  3. Was it intuitive or confusing to operate this particular machine?
  4. Do you view needing pods in order to use the machine as an advantage or disadvantage? Why?
  5. How often would you use this machine?
  6. When would be the first time in the day that you would use this machine?

Sketches

Observations

Person A
  • Understands to plug in and press power button
  • Put pod inside before pressing brew button
  • However, forgot to add water before brewing
  • Turned off, then added water to container before pressing brew button
  • Was able to successfully brew coffee
Question Responses
  1. First time use
  2. To make coffee at home instead of needing to buy; cheaper alternative
  3. Compared to other machines, this machine was more intuitive due to less buttons and fewer possible actions; however, wished for more labels to add clarity
  4. Disadvantages because bad for environment and only has single use - powdered version can have multiple uses and takes up less space; can only make one cup of coffee at a time
  5. Every day
  6. Morning
Person B
  • Confused with buttons and their function
  • Pressed brew button multiple times, but machine wasn't functioning; figured out that the water container lid had to be closed in order to start brewing
  • Was able to successfully brew coffee
Question Responses
  1. First time use
  2. Convenience of having quick access to coffee
  3. Was confusing because buttons didn't do what they expected them to do; mental model did not line up with machine's true functionality
  4. Advantage because of easiness and allows for quicker variety
  5. Several times a day
  6. Morning
Person C
  • Quickly got through each step
  • Demonstrated knowledge of how to operate machine
  • Was able to successfully brew coffee
Question Responses
  1. Has used in the past
  2. For easy energy boost whenever needed
  3. Intuitive because they have used the machine in the past
  4. Disadvantage because it's difficult to make large quantities of coffee in a short amount of time with pods; however; convenience is nice
  5. Usually once a day, but if has lots of work, multiple times a day
  6. Morning
